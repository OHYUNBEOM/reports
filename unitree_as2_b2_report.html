B2 vs As2 — 가장 큰 차이점 정리

두 라인업은 용도와 운용 철학 자체가 다릅니다

B2의 강점

  • IP67 완전 방수·방진 — 폭우, 분진, 수중 투입 가능
  • 360 N·m 토크 + 120 kg 적재 — 중량 장비 운반 가능
  • 6 m/s+ 최고 속도 — As2 대비 약 1.2~2배
  • 45° 이상 경사 등판 — 산업 현장 극한 지형 대응
  • 멀티 컴퓨팅(i5+i7×3+Orin NX) 고성능 처리

As2의 강점

  • 18 kg 경량 — B2의 1/3, 1인 운용 및 이동 용이
  • ISS 3.0 지능형 팔로우 — 센티미터 정밀도 추종
  • 스피커 + 마이크 탑재 — 음성 상호작용 가능
  • 전방 조명 탑재 — 야간 순찰·시야 확보
  • Wi-Fi 6 / BT 5.2 / GPS / 4G 무선 통신 내장
  • EDU 모델: SDK + Jetson + 자동 충전 도크 → 2차 개발 최적
  • UniStore 앱 마켓으로 기능 즉시 확장
핵심 요약: B2는 "극한 환경에서 무거운 임무를 수행하는 산업용 대형 로봇"이고, As2는 "가볍고 스마트하며 확장성이 뛰어난 차세대 범용 로봇"입니다. B2가 힘과 내구성에서 압도적인 반면, As2는 통신·음성·팔로우·조명 등 스마트 기능과 개발 생태계에서 확실한 우위를 가집니다.

SDK — 로봇을 자유롭게 제어하는 개발 도구

SDK(Software Development Kit)란? 로봇에 새로운 기능을 추가하거나, 원하는 대로 동작을 프로그래밍할 수 있게 해주는 개발 도구 모음입니다. 스마트폰 앱을 만들 때 개발 도구가 필요한 것처럼, 로봇의 커스텀 기능을 만들기 위해서 SDK가 필요합니다.

SDK 공통 프레임워크의 장점

  • Unitree 전 로봇과 호환: As2용으로 만든 프로그램이 B2, Go2, H1 등 다른 Unitree 로봇에서도 최소한의 수정으로 작동합니다
  • 개발 도구를 한 번만 배우면 됨: 하나의 SDK 사용법을 익히면 Unitree의 모든 로봇 라인업에 적용할 수 있어 효율적입니다
  • 기존 개발 사례 활용 가능: 이미 다른 Unitree 로봇 사용자들이 만들어둔 코드와 노하우를 그대로 참고할 수 있습니다
  • 글로벌 표준 기반: 산업용 로봇 통신 표준(CycloneDDS)과 로봇 운영체제(ROS 2)를 기반으로 하여, 범용적인 로봇 소프트웨어 생태계와 연동됩니다

SDK로 할 수 있는 것

  • 로봇의 걷기, 달리기, 방향 전환 등 움직임을 직접 제어
  • 카메라, LiDAR 등 센서 데이터를 실시간으로 수집
  • AI를 연동하여 자율 판단·행동하는 기능 개발
  • 원격 제어 프로그램 개발
  • 자동 순찰 경로 설정 및 실행

개발 도구 2종 비교

  • C++ 도구: 로봇을 매우 빠르고 정밀하게 제어할 때 사용. 상용 제품에 적합
  • Python 도구: AI 기술(이미지 인식, 음성 처리 등)과 연동이 쉬움. 시제품 제작·실험에 적합
  • 두 도구 모두 동일한 기능을 제공하며, 상황에 맞게 선택 가능

UniStore — 로봇 앱 마켓플레이스

UniStore란? 스마트폰의 앱스토어처럼, Unitree 로봇에 다양한 기능을 앱 형태로 설치할 수 있는 플랫폼입니다. As2 전 모델(AIR/PRO/EDU)에서 사용할 수 있습니다.

2차 개발이란?

Unitree가 만든 로봇(플랫폼)을 기반으로, 우리가 원하는 추가 기능이나 솔루션을 직접 개발하는 것을 말합니다. 예를 들어, As2 로봇에 디스펙터 엣지 디바이스를 연결하고, 자율 순찰 + AI 점검 기능을 추가 개발하는 것이 2차 개발에 해당합니다.

As2 EDU가 2차 개발을 공식 지원한다는 것의 의미

  • 공식 허가: EDU 모델 구매 시, Unitree가 제공하는 개발 도구(SDK)를 사용하여 자유롭게 커스텀 프로그램을 만들 수 있습니다
  • 하드웨어 확장 자유: 엣지 디바이스, 추가 센서 등 외부 장비를 연결하여 기능을 확장할 수 있습니다
  • 소스 코드 공개: 개발 도구의 소스 코드가 GitHub에 공개되어 있어, 내부 동작을 파악하고 필요에 맞게 수정할 수 있습니다
  • 앱 유통 채널: 개발한 솔루션을 UniStore를 통해 공식적으로 판매·유통할 수 있습니다
참고: AIR/PRO 모델은 2차 개발을 공식 지원하지 않습니다. 2차 개발이 필요한 경우 반드시 EDU 모델을 선택해야 합니다.

Section 05

대표님 요청 사항 답변

요청 사항에 대한 분석 결과

아래는 대표님께서 요청하신 주요 검토 사항에 대한 정리입니다.

요청 2 As2의 주요 특징 분석 — IP 등급, CPU, ISS 3.0 지능형 팔로우 시스템, 스피커, 마이크, 전방 조명
  • IP 등급: As2 PRO/EDU는 IP54(생활 방수·방진), AIR는 미지원. B2의 IP67(완전 방수) 대비 등급이 낮으나, 일반 옥외 환경에서는 IP54로 충분합니다. 극한 환경(폭우·분진)에서는 B2가 필수입니다.
  • CPU: As2는 Qualcomm 8Gen 기반 프로세서를 사용하며, EDU 모델은 NVIDIA Jetson Orin 시리즈를 추가 탑재하여 로봇 위에서 직접 AI 추론이 가능합니다. B2는 Intel i5×1 + i7×3 + Jetson Orin NX 조합으로 원시 연산 능력은 더 높습니다.
  • ISS 3.0 (Intelligent Side-follow System): As2에만 탑재된 핵심 차별 기능으로, UWB + 비전 퓨전 방식으로 사람을 센티미터 수준의 정밀도로 추종합니다. 측면·후면 팔로잉까지 지원하여, 산업 현장 안내·물류 수행·보안 수행 등에 바로 활용 가능합니다.
  • 스피커 + 마이크: As2 전 모델에 탑재. 음성 안내 방송, 현장 경고 알림, 양방향 음성 통신이 가능합니다. AI 음성 인식과 결합하면 음성 명령 제어까지 구현할 수 있습니다. B2에는 미탑재.
  • 전방 조명: As2 전 모델에 탑재. 야간 순찰 시 카메라 보조광, 위험 구역 시야 확보, 존재감 표시(보안 억제 효과) 등으로 활용됩니다. B2에는 미탑재.
요청 3 개발자 리소스 분석 (As2 EDU) — SDK 공통 프레임워크 활용, UniStore 플랫폼 연동 시 이점
  • SDK 공통 프레임워크: Unitree SDK2(C++ / Python)는 CycloneDDS + ROS 2 기반의 통합 개발 도구입니다. As2 EDU용으로 작성한 코드가 B2, Go2, H1 등 Unitree 전 라인업에서 최소 수정으로 재사용 가능하여, 한 번 투자한 개발 역량이 전체 제품군에 확장됩니다.
  • SDK로 가능한 것: 관절 모터 직접 제어, 센서(카메라·LiDAR) 데이터 실시간 수집, AI 자율 판단 기능 개발, 원격 제어 프로그램 구현, 자동 순찰 경로 프로그래밍 등이 가능합니다.
  • UniStore 연동 이점: 스마트폰 앱스토어처럼 로봇에 기능을 앱 형태로 설치하는 플랫폼입니다. 자체 개발한 솔루션을 UniStore에 등록하면 글로벌 유통 채널을 즉시 확보하게 됩니다. 또한 이미 등록된 순찰·점검·AI 인식 등의 앱을 활용하면 개발 기간과 비용을 절감할 수 있습니다.
  • 2차 개발 공식 지원: EDU 모델만 Unitree가 공식적으로 2차 개발을 허가합니다. Jetson 확장 포트, 충전 도크, SDK 전체 접근권이 포함되어 있어, 커스텀 솔루션 사업에는 EDU가 필수 선택입니다.
  • 실질적 사업 효과: SDK 공통 프레임워크 + UniStore 생태계를 활용하면, 소수 개발 인력으로 빠르게 시제품을 만들고 → UniStore로 유통하며 → 향후 다른 Unitree 로봇에도 확장하는 전략이 가능합니다.
