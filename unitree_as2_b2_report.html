SDK 공통 프레임워크 — CycloneDDS 기반

핵심 포인트: unitree_sdk2는 Go2, B2, H1 등 Unitree 전 로봇 라인업과 동일한 공통 프레임워크(CycloneDDS)를 사용합니다. As2 EDU에도 동일하게 적용됩니다.

CycloneDDS란?

  • Eclipse Foundation의 오픈소스 DDS(Data Distribution Service) 구현체
  • 산업용 실시간 통신 미들웨어 표준 (OMG DDS 규격)
  • ROS 2 (Robot Operating System 2)의 공식 통신 레이어로 채택

SDK 공통 프레임워크 활용의 의미

  • 코드 재사용성: B2용으로 개발한 코드를 As2에 최소 수정으로 적용 가능 (반대도 동일)
  • 학습 곡선 최소화: 하나의 SDK를 익히면 Unitree 전 로봇 라인업에 적용 가능
  • 생태계 공유: 다른 Unitree 로봇 사용자들의 개발 사례, 코드, 노하우를 그대로 활용 가능
  • ROS 2 호환성: CycloneDDS 기반이므로 ROS 2 생태계의 방대한 패키지와 도구 활용 가능

unitree_sdk2 (C++) — 고성능 실시간 제어

  • 저수준 모터 제어: 12개 관절의 위치/속도/토크를 직접 제어 (최대 500Hz)
  • 고수준 모션 제어: 속도 명령, 자세 명령, 보행 모드 전환, 내장 액션 트리거 등 추상화 API
  • 센서 데이터 수신: IMU, 발 힘 센서, 관절 상태, 배터리 등 DDS 토픽 실시간 수신
  • 안전 메커니즘: 워치독 타이머, 안전 제한 기능 내장

unitree_sdk2_python — 빠른 프로토타이핑

  • C++ SDK와 동일한 DDS 토픽 및 메시지 타입 사용
  • Python 생태계 즉시 연동: PyTorch, TensorFlow, OpenCV, ROS 2
  • 낮은 진입 장벽: 연구자, 교육자, AI 개발자 빠른 시작
  • 풍부한 예제: 로봇 상태 읽기, 속도 명령, 관절 제어 등
비앤비 개발 시 권장: 프로토타이핑은 Python SDK로 빠르게 검증 → 상용 배포 시 C++ SDK로 전환하여 성능 최적화

UniStore 플랫폼 연동 효과

UniStore란? Unitree가 운영하는 로봇 앱 마켓플레이스로, 스마트폰 앱스토어와 유사한 개념입니다. As2 전 모델(AIR/PRO/EDU)에서 지원합니다.

2차 개발 허가 범위

As2 EDU 2차 개발 지원의 의미

  • 공식 허가: EDU 모델 구매 시 SDK를 통한 커스텀 소프트웨어 개발이 공식 허용됨
  • 플랫폼 + 확장 = 자유: 추가 하드웨어(엣지 디바이스 등) 연결 및 커스텀 소프트웨어 개발이 자유로움
  • SDK 오픈소스: GitHub 공개 소스 코드로 내부 동작 원리 파악 및 필요에 맞게 수정 가능
  • UniStore 유통: 개발 솔루션을 UniStore를 통해 공식 유통 채널 확보
참고: AIR/PRO 모델은 2차 개발을 공식 지원하지 않습니다. 비앤비의 2차 개발 목적이라면 반드시 EDU 모델을 선택해야 합니다.
