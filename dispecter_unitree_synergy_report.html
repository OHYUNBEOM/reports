Section 01

디스펙터 회사 개요

"Accelerating the Era of Physical AI" — 피지컬 AI 시대를 가속하는 로봇 워크플로우 플랫폼 기업

핵심 제품: Cerebix α (세레빅스 알파)

디스펙터의 핵심 제품은 Plug-n-Play 로봇 워크플로우 플랫폼으로, 3가지 요소로 구성됩니다.

Cerebix Core

사족보행 로봇 등에 탑재하는 피지컬 AI 엣지 디바이스. NVIDIA Jetson AGX Orin 기반, 4kg, 다종 센서/매니퓰레이터 연동

Cerebix Copilot

다수 로봇의 원격제어, 미션 자동화, 다중 로봇 협업 운용, 데이터 분석/보고를 위한 개방형 워크플로우 플랫폼

스타터 앱 3종

자율 경비, 공정설비 점검, 현장 안전관리 등 즉시 배포 가능한 산업용 앱

디스펙터의 핵심 강점

하드웨어 독립적 (Vendor-Agnostic)
특정 로봇 제조사에 종속되지 않는 플랫폼
모듈화 (Modular)
기능을 모듈 단위로 추가/확장 가능
확장성 (Scalable)
1대에서 다수 로봇 Fleet 운영까지 확장
누적적 지능 (Cumulatively Skilled)
AI 스킬이 지속적으로 고도화

투자 및 실적

  • Seed 투자 4.8억원 (2023, 더인벤션랩)
  • Pre-A 18억원 투자유치 (2025.4, 더인벤션랩, 스케일업파트너스)
  • 딥테크 TIPS 15억원, 스케일업 TIPS 11.4억원
  • 레인보우로보틱스와 MoU 체결 (2025.9)
  • 다수 고객사에 솔루션 납품 중 (영남대병원, GS건설 등)

As2 EDU와 Cerebix Core의 하드웨어 결합 구조 및 소프트웨어 통합 시너지를 분석합니다.

하드웨어 결합 구조

dSPECTER — Cerebix Core (엣지 HW)

자율주행, 객체 인식, 경로 계획

기술적 호환성 분석

항목 As2 EDU 제공 Cerebix 요구 호환성
엣지 컴퓨팅 Jetson Orin NX (선택) Jetson AGX Orin (자체 탑재) Cerebix Core가 자체 GPU를 갖고 있으므로 As2의 Orin NX 없이도 가능. 둘 다 사용 시 이중 AI 파이프라인 구성 가능
적재 능력 15kg 연속 보행 Cerebix Core 4kg + 추가 센서 충분. 센서 포함 약 6~8kg으로 여유 있음
통신 이더넷 ×1, Wi-Fi 6, BT 5.2 이더넷/Wi-Fi 연동 호환 가능
전원 BAT 전원 출력 72W 필요 As2 배터리(648Wh)에서 공급 시 운행 시간 단축 예상. 확인 필요
SDK unitree_sdk2 (CycloneDDS) 자체 미들웨어 CycloneDDS/ROS 2 기반 통합 가능
물리적 탑재 컴팩트 본체 (720mm) 외부 부착 또는 내부 탑재 상부 탑재 가능. 다만 B2(대형) 대비 공간 제약 있음

LiDAR + 카메라 실시간 지도 생성 · 공유 SLAM 기반 자율 내비게이션 구현
스피커 / 마이크 자연어 기반 미션 지시 음성 명령 → 자율 미션 수행 파이프라인
